CliniTrace is a doctor-led, AI-assisted visual monitoring platform focused on:
- **Augmentation** (not diagnosis)
- **Structured clinical documentation**
- **Longitudinal tracking** (progression over time)
- **Quality-gated image capture** for safer, more consistent assessments

This repo contains the **public website + documentation library** and is intended to be published via **GitHub Pages**.
